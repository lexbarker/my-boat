# Race-Boat

Enables a competitor to create a boat/crew configuration known as boat, for use by the other apps in the suite. Each competitor can create multiple boat entries.

## Boat Data

Data shall be as JSON
```javascript
{
name: str,
boat: str,
class: str,
sailNumber: int,
pyNum: int,
helm: str,
crew: str,
clubId: int,
}
```
