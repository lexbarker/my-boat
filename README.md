# my-boat

Enables a racer to create a boat/crew configuration for use by the other apps in the suite. Each club member can create multiple boat entries.

## Boat Data

Data shall be as JSON
```javascript
{
	name: str,
	class: str,
	sailNumber: int
	pyNum: int,
	helm: str,
	crew: str,
	clubId: int,
}
```
